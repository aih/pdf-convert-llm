About the Commonwealth Legislative Drafting Manual

This site provides a navigable web display of the Commonwealth Legislative Drafting Manual. The Manual provides expert guidance to legislative drafters throughout the Commonwealth.

This is an experimental transformation of the official Commonwealth Legislative Drafting Manual from PDF format to HTML for web display.

⚠️ Experimental Translation: This feature uses the browser's built-in Translation API (currently supported in Chrome 141+). Any language other than English represents automatically translated text and is not official. The translation may contain inaccuracies or errors. For authoritative information, please refer to the original English version. Additional languages can be added upon request.
